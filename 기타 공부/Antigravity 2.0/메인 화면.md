## 1. 상단 메뉴바 (Top Bar)

Antigravity : 버전, 업데이트 확인용

File : 
- `New Conversation` : 새 대화 시작
- `Create Project` : 새 프로젝트 생성
- `Command Palette` : 명령어 팔레트 열기

View : 화면 확대, 축소
Window : 창버튼 (최대, 최소, 닫기)

## 2. 왼쪽 사이드바 (Left Sidebar)

- **`+ New Conversation`:** 새로운 AI 대화를 시작할 수 있는 메인 버튼.

- **`Conversation History`:** 이전 대화 기록을 확인할 수 있는 메뉴.

- **`Scheduled Tasks`:** 백그라운드에서 실행되도록 예약된 작업이나 자동화 워크플로우를 관리하는 메뉴.

- **`Projects`:** 프로젝트 리스트를 보여주는 섹션으로, 우측의 아이콘들을 통해 필터링하거나 새 프로젝트 폴더를 추가가능.
- **⚙️ `Settings`:** 최하단에 위치한 환경 설정 메뉴.


### 1. New Project (새 프로젝트)

- **역할:** 대화를 시작함과 동시에, 새로운 작업 폴더를 지정하거나 신규 프로젝트 공간을 생성.

- **용도:** 기존에 만들어둔 로컬 소스코드 폴더를 Antigravity에 새로 등록하여 에이전트에게 전체 분석을 맡기거나, 완전히 새로운 독립된 앱 개발을 시작할 때 선택합니다. 여기에 등록하면 왼쪽 사이드바의 `Projects` 목록에 새 프로젝트가 추가됩니다.

### 2. Quick Start (빠른 시작)

- **역할:** 복잡한 프로젝트 폴더 설정 과정 없이, 기본적인 뼈대 코드(Template)나 임시 샌드박스 환경을 자동으로 구성하여 즉시 개발 작업을 시작하는 기능.

- **용도:** 특정 프레임워크나 언어로 간단한 프로토타입을 빠르게 구현해 보거나, AI 에이전트의 성능을 가볍게 테스트해보고 싶을 때 유용합니다.

### 3. No Project (프로젝트 없음)

- **역할:** 특정 소스코드 폴더나 작업 공간을 전혀 연결하지 않고, 완전히 독립된 대화 세션을 엽니다.

- **용도:** AI 에이전트가 내 컴퓨터의 파일을 읽거나 수정할 필요가 없을 때 선택합니다. 단순한 알고리즘 개념 질문, 코딩 문법 확인, 일반적인 개발 지식을 물어볼 때 적합하며, 불필요한 파일 정보를 불러오지 않아 실행이 가볍습니다.
