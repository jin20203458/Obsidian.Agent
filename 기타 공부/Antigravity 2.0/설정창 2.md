
## App Settings(앱 설정 관리)
![[Pasted image 20260527164026.png]]
### 1. General (일반 앱 설정)

- **Prevent Sleep (컴퓨터 절전 모드 방지):**
    
    - **역할:** Antigravity 앱이 켜져서 동작하는 동안에는 **컴퓨터가 자동으로 절전 모드(화면 꺼짐, 대기 모드)로 들어가는 것을 막아주는** 토글 스위치입니다. 
    
    - **왜 필요할까요?** 최신 에이전트 스택인 2.0 버전은 백그라운드에서 수십 분 동안 복잡한 코드 설계나 테스트 작업을 스스로 수행하곤 합니다. 그런데 도중에 컴퓨터가 잠들어버리면 작업이 끊기겠죠? 긴 백그라운드 자동화 작업을 걸어놓을 때 켜두면 아주 유용한 옵션입니다.
    
- **Keep In Menu Bar (메뉴 바/트레이에 유지):**
    
    - **역할:** 앱의 창(X 버튼)을 닫더라도 프로그램을 완전히 종료하지 않고, **작업 표시줄(윈도우의 트레이 아이콘 영역 또는 맥의 상단 메뉴 바)에 상주하며 백그라운드에서 계속 실행**되도록 만드는 토글입니다.
    
    - **왜 필요할까요?** 화면을 넓게 쓰기 위해 창은 닫아두되, 백그라운드에서 에이전트가 돌려놓은 예약 작업(`Scheduled Tasks`)이 계속 정상 작동해야 할 때 활성화합니다.


### 2. Notifications (알림 설정)

- **Notification Settings (알림 제어):**
    
    - 에이전트가 작업을 완료했거나 디버깅 중 에러가 발생했을 때, OS 바탕화면 우측 하단에 **푸시 알림**을 띄울지 제어하는 메뉴입니다.
        
    - 오른쪽에 있는 **`Open System Preferences`** 버튼을 누르면 Antigravity 앱 내부가 아니라, 현재 쓰고 계신 윈도우(또는 맥)의 **운영체제 시스템 알림 설정 창**이 열려 권한을 직접 켜고 끌 수 있게 연결해 줍니다.

## 1. Folders (연동된 로컬 폴더 및 브랜치)

- **`intelligent-lavoisier/ 🌿 master`:** * 현재 이 프로젝트가 내 컴퓨터의 어떤 폴더를 바라보고 있는지, 그리고 Git 기반의 어떤 브랜치(`master`)에 연결되어 있는지 보여줍니다.
    
    - 하단의 `+ Add Folder`를 누르면 이 프로젝트 안에 다른 서브 폴더들을 추가로 연동하여 AI 에이전트가 함께 분석하게 만들 수 있습니다.

## 3. Agent Behavior (에이전트 행동 가이드라인)

- **Artifact Review Policy (산출물 검토 정책):**
    
    - **중요 개념:** 에이전트가 코딩하다가 완성한 독립된 산출물(HTML 문서, 완성된 스크립트 코드, 마크다운 기획서 등)을 사용자 화면에 띄워줄 때, "주인님, 이거 다 만들었는데 한 번 검토해 보실래요?"라고 물어보는 행동 지침입니다.
        
    - 현재 `Always Ask`로 되어 있으므로, AI가 독단적으로 결과물을 밀어 넣지 않고 무조건 나에게 먼저 검토 팝업을 띄우도록 안전하게 세팅되어 있습니다.
        

## 4. Local Permissions (이 프로젝트 전용 로컬 권한)

여기가 바로 전역 권한보다 우선하는 **치트키 구역**입니다. 안내문에도 적혀있듯 여기서 설정한 값들이 Global Settings보다 우선순위(Higher Priority)를 가집니다.

- **File Access Rules / Network Access Rules / Terminal Commands / Commands Outside Sandbox / MCP Tools:**
    
    - 기억나시죠? 전역 설정에서 컴퓨터 전체 보안을 위해 '터미널 명령어 실행'을 막아두었더라도, 이 안전한 `intelligent-lavoisier` 프로젝트 내에서만큼은 오른쪽 `Open`을 눌러 `Allow(허용)`로 바꿔주면 AI가 이 폴더 안에서는 자유롭게 패키지를 설치(`npm install` 등)하고 코드를 테스트할 수 있게 됩니다.
        

## 5. Customizations (이 프로젝트 전용 토큰 버젯)

- **역할:** 전역 커스텀 설정과 별개로, **오직 이 `intelligent-lavoisier` 프로젝트를 다룰 때만 적용할 전용 규칙(Rules)이나 전용 MCP 서버**가 차지하는 기억 용량을 보여줍니다.
    
- **현재 상태:** 역시 등록된 로컬 커스텀이 없어서 `100.0% available`로 뇌 용량이 아주 온전하고 쾌적하게 비어있습니다.
