
현재 사용하실 수 있는 슬래시 명령어(shortcuts)들은 다음과 같습니다. 각 명령어의 역할과 추천하는 상황은 다음과 같습니다.

| 명령어                     | 설명 및 추천 상황                                                                                         |
| :---------------------- | :------------------------------------------------------------------------------------------------- |
| **`/goal`**             | **장기 실행 태스크 처리**<br>밤새 또는 장시간 동안 에이전트가 중단 없이 매우 철저하게 목표를 완수해야 하는 긴 작업이 있을 때 사용합니다.                 |
| **`/schedule`**         | **스케줄 및 타이머 설정**<br>특정 작업을 주기적으로 반복(cron schedule)해서 실행하거나, 일회성 타이머를 설정하여 예약 작업을 수행하고 싶을 때 사용합니다.  |
| **`/browser`**          | **웹 브라우징 및 검색**<br>웹 페이지 방문, 검색, 혹은 웹 애플리케이션과의 상호작용이 필요한 태스크를 수행할 때 추천합니다.                         |
| **`/teamwork-preview`** | **멀티 에이전트 협업**<br>프로젝트 규모가 크고 복잡하여, 여러 명의 자율적인 하위 에이전트들(subagents)이 팀을 이루어 함께 협업하는 것이 유리할 때 사용합니다. |
| **`/grill-me`**         | **인터랙티브 설계 인터뷰**<br>질의응답을 통해 요구사항을 구체화하고 최적의 설계 결정을 내려 구현 계획을 에이전트와 맞추고 싶을 때 사용합니다.                |

필요한 작업의 성격에 맞춰 채팅창에 직접 입력하여 실행해 보실 수 있습니다.
